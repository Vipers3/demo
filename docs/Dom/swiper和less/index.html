swiper 和 less

1、swiper 插件​

官方文档​

<!DOCTYPE html>
<html lang="zh">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <script src="./js/swiper-3.4.2.min.js"></script>
    <link rel="stylesheet" href="./js/swiper-3.4.2.min.css" />
    <style>
      .swiper-container {
        width: 600px;
        height: 300px;
      }
    </style>
  </head>
  <body>
    <div class="swiper-container">
      <div class="swiper-wrapper">
        <div class="swiper-slide">Slide 1</div>
        <div class="swiper-slide">Slide 2</div>
        <div class="swiper-slide">Slide 3</div>
      </div>
      <!-- 如果需要分页器 -->
      <div class="swiper-pagination"></div>

      <!-- 如果需要导航按钮 -->
      <div class="swiper-button-prev"></div>
      <div class="swiper-button-next"></div>

      <!-- 如果需要滚动条 -->
      <div class="swiper-scrollbar"></div>
    </div>
    <!-- 导航等组件可以放在container之外 -->
    <script>
      /*
            1.引入插件的css和js
        */
      var mySwiper = new Swiper(".swiper-container", {
        // swiper的滑动方向，可设置为水平方向切换 horizontal 或垂直方向切换 vertical
        direction: "horizontal",
        loop: true,

        // 如果需要分页器
        pagination: ".swiper-pagination",

        // 如果需要前进后退按钮
        nextButton: ".swiper-button-next",
        prevButton: ".swiper-button-prev",

        // 如果需要滚动条
        scrollbar: ".swiper-scrollbar",
      });
    </script>
  </body>
</html>

2、less 预编译器​

less 的使用​

  • less 可以赋予 css 逻辑处理的能力

  • less 的使用（单独使用）

    • 查看 less 有没有安装好 ：lessc -v
    • 通过 lessc 编译 less 文件： 把 lessc 编译成 css 文件 最后再执行
    • 指令：lessc less 文件路径 目标 css 文件的路径
    • 注意：命令行地址的目录

less 里的注释​

  • 1.less 的注释（less 语言的注释，这个注释不会被编译出来）
  • 2.书写 css 注释（css 注释会被 less 编译到 css 文件里去）
  • 3.如果使用了 less 如果需要修改 css 样式，不要直接修改 css 文件，不然会被 less 的重新编辑覆盖掉

嵌套​

普通嵌套​

.container {
  width: 500px;
  height: 500px;
  background: @a;
  .box1 {
    width: 100px;
    height: 100px;
    background: @b;
  }
  .box2 {
    width: 100px;
    height: 100px;
    background: @c;
  }
}

嵌套里可以通过&符号引用父级​

.box1 {
  width: 300px;
  height: 300px;
  background: @a;
  // 直接通过&符号引用父级的类名
  &:hover {
    background: @c;
  }
}

变量的定义和使用​

  • 定义变量 作为样式名使用
  • 注意：需要帮变量放在插值表达式里执行@{变量名}
@w:width;
@mydiv:.box3;
@myw:100px;
@myh:100px;
@{mydiv}{ //属性名的时候需要放在插值表达式里@{变量}
    @{w}: @myw;
    height: @myh;
}
// 地址是组合 需要拼接的时候 那么变量也需要放在插值表达式里
@myurl: "./1.png";
.box4 {
  width: 200px;
  height: 200px;
  background-image: url("mydir/@{myurl}");
}

less 里的运算​

注意：运算时至少要有一个单位

加法运算​

@a: 10px;
@b: 20;
.box {
  width: @a + @b;
}

减法运算 前后需要加空格​

@a: 10px;
@b: 20px;
.box {
  width: @a - @b;
}

乘法运算​

@a: 10px;
@b: 20px;
.box {
  width: @a * @b;
}

除法运算​

// 注意：除法需要加括号
@a: 10px;
@b: 20px;
.box {
  width: (@a / @b);
}

混入模式​

普通函数​

.common {
  font-size: 20px;
  color: black;
  border: 1px solid red;
}

.box {
  .common; //调用函数
  width: 100px;
  height: 100px;
}

带参数的函数​

// 定义有参数的函数
.common(@color) {
  font-size: 20px;
  color: black;
  border: 1px solid;
  background: @color;
}
// 调用函数 .box1的背景颜色是蓝色
.box1 {
  width: 100px;
  height: 100px;
  .common(blue);
  // 调用的时候可以传入实参
}
// 调用函数 .box2的背景颜色是黄色
.box1 {
  width: 100px;
  height: 100px;
  .common(yellow);
  // 调用的时候可以传入实参
}

带默认参数的函数​

// 给@h 变量和@color变量设置默认值，如果有值传入 按照传入的值为准，如果没有值传入那么就按默认值来赋值
.common(@w,@h:50px,@color:blue) {
  width: @w;
  height: @h;
  background: @color;
}
.box1 {
  .common(100px, 200px, red);
  border: 1px solid #fff;
}
.box2 {
  .common(@w, @h, @color);
}

函数作用域​

.container {
  width: 100px;
  height: 100px; /*  */
  .common(@color) {
    background: @color;
    border: 1px solid red;
  }
}
.box1 {
  .container .common(blue); //全局作用域范额为内没有common这个样式 所以获取不到
  width: 300px;
  height: 300px;
}

匹配模式​

  • 类似 switch, @_是一个固定写法
.container(@_) {
  width: 100px;
  height: 100px;
}

// 情况一:case color1
.container(color1) {
  color: blue;
}

// 情况二:case color2
.container(color2) {
  color: red;
}

.box {
  .container(color1);
  border: 1px solid red;
}

重载​

  • 根据不同参数的个数加载不同的函数
// 定义一个common，只接收一个参数
.common(@w) {
  width: @w;
  height: 100px;
}
.common(@w,@color) {
  width: @w;
  height: 200px;
  background: @color;
}

.box1 {
  .common(100px);
  border: 1px solid #fff;
}
.box2 {
  .common(100px,red);
  border: 2px dotted #bfc;
}

条件判断​

条件判断​

.common(@num) when (@num>20) {
  width: 100px;
  background: red;
}
.common(@num) when (@num<20) {
  width: 10px;
  background: blue;
}

.box1 {
  .common(50);
}

通过 and 链接多个条件​

.common(@num) when(@num>20) and(@num<100) {
  width: 80px;
  height: 80px;
  background: red;
}

.box {
  .common(10);
}

通过 或者关系链接条件 通过逗号链接多个条件​

.common(@num) when (@num<=20),(@num>=100) {
  height: 200px;
  width: 200px;
  background: #bfa;
}

less 会提供各种预定义函数​

官方文档

// 向上取整
@a: 100px;
@b: 3px;
.box1 {
  width: ceil((@a / @b));
  height: 100px;
}

// 转换布尔值
@flag:boolean (2<5);
.box {
  background: if((@flag) "red", "blue");
}

样式的模块化​

  • less 里可以引入其他文件
  • 可以引入 css 的文件 @import url("./a.css");
  • 可以引入 less 文件 @import url("./b.less");
