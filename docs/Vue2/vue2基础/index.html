vue2 基础

vue 简介​

vue 是一套用来构建用户界面的渐进式 JavaScript 框架​

Vue 的核心的功能是一个视图模板引擎，在声明式渲染（视图模板引擎）的基础上，我们可以通过添加组件系统、客户端路由、大规模状态管理来构建一个完整的框架。更重要的是，这些功能相互独立，你可以在核心功能的基础上任意选用其他的部件，不一定要全部整合在一起。可以看到，所说的“渐进式”，其实就是 Vue 的使用方式，同时也体现了 Vue 的设计的理念

vue 的特点​

  • 声明式编码，让编码人员无须操作 DOM。提高开发效率
  • 组件化编码，提高代码复用率、且让代码更好的维护
  • 使用虚拟DOM+Diff算法，实现真实 DOM 的最小化更新（也就是最大化复用）
  • 编码简洁, 体积小, 运行效率高, 适合移动/PC 端开发
  • 它本身只关注 UI, 可以轻松引入 vue 插件或其它第三库开发项目
  • 遵循MVVM模式

MVVM​

M: Model 模型 => 包含 n 个可变数据的对象 => 在 vue 中就是 data 数据对象

V: View 视图 => 动态显示数据的页面 => 在 vue 中就是模板产生的页面

VM: ViewModel 视图模型 => 是 M 与 V 相互通信的"桥梁" => 在 vue 中就是 vue 实例对象

一方面, 它监视 data 数据变化后, 动态更新页面 另一个方面, 还可以监视用户输入(dom 事件), 将最新输入的值保存到 data 中

vue2 的基本使用​

<script src="https://cdn.bootcdn.net/ajax/libs/vue/2.6.12/vue.js"></script>
<div id="app">
    <h1>
        hello {{msg}}
    </h1>
</div>
<script>
    // 引入vue.js 全局就会有一个函数Vue
    const vm = new Vue({
        //配置对象：属性名固定的对象
        el: "#app", //element元素，用来指定当前的Vue实例服务于哪一个容器，值为css选择器字符串
        data: { //数据：用来渲染到页面中，供el指定的容器使用，暂时写一个对象，如果在组件中我们会写成函数
            msg: "world",
        }
    })
    console.log(vm);//返回vue的实例
</script>

手动挂载​

  • 如果 Vue 实例在实例化时没有收到 el 选项，则它处于“未挂载”状态，没有关联的 DOM 元素。可以使用 vm.$mount() 手动地挂载一个未挂载的实例。
  • new Vue 时，el 和$mount 并没有本质上的不同。
<div id="app"></div>
<template id="tmpl">
  <div class="box">{{msg}}</div>
</template>
<script>
  const p = new Vue({
    data: {
      msg: "今天天气真不错",
    },
    template: "#tmpl", //用来指定模板
  });
  setTimeout(() => {
    p.$mount("#app");
  }, 1000);
</script>

Object.defineProperty​

给对象添加带 getter 与 setter 的属性 也可以修改

  • getter 回调函数: 时机: 当读取属性时调用, 作用: 计算返回属性值, this: 当前对象
  • setter 回调函数: 时机: 当修改属性值时调用, 作用: 监视属性值的变化的, this: 当前对象

vue2 中实现数据代理与数据劫持的核心语法

let obj = {
  firstName: "A",
  lastName: "B",
};
Object.defineProperty(obj, "fullName", {
  get() {
    return this.firstName + "-" + this.lastName;
  },
  set(val) {
    const name = val.split("-");
    this.firstName = name[0];
    this.lastName = name[1];
  },
});
obj.firstName = "C";
obj.lastName = "D";
console.log(obj.fullName); //读取fullName的值时，会调用get方法
obj.fullName = "E-F"; //修改fullName的值时，会调用set方法
console.log(obj.firstName);

vue2 代理​

原理：通过 Object.defineProperty 将 data 中的所有属性添加到 vm 实例上，并指定 get 和 set 方法代理 data 中的属性，当访问 vm 实例上的属性时，就会调用 get 方法，return this.data 上的同名属性,修改 vm 实例上的属性时，就会调用 set 方法修改 this.data 上的同名属性

 <script>
   function Vue(options) {
     this._data = options.data;
     Object.keys(this._data).forEach((item) =>
       Object.defineProperty(this, item, {
         get() {
           console.log("getter");
           return this._data[item];
         },
         set(val) {
           console.log("setter");
           this._data[item] = val;
         },
       });
     });
   }
 </script>
 <script>
   const vm = new Vue({
     data: {
       msg: "今天天气真不错",
       msg2: "嘿嘿嘿",
     },
   });
   console.log(vm.msg);
   vm.msg = "小白";
   console.log(vm.msg);
 </script>

vue 中的插值语法​

vue 中的 template 可以使用 vue 的插值语法, 动态显示表达式的值，表达式会读取 vm 实例上的对应属性，也就是被代理的 data 中的数据

vue 的插值语法支持 JS 的所有数据类型 在显示时, 会自动将数据转换为字符串值显示

特别的:undefined/null 显示空白 react 也是这样的,对象/函数也是支持 react 不支持,bigint/symbol 也是支持

class 的动态绑定​

  • 情况 1: 类名是确定的且只有一个, 只是不确定有没有 => 三元写法 :class="hasA ? 'classA' : ''"
  • 情况 2: 类名是确定的且是多个, 只是不确定有没有 => 对象写法 :class="{classA: hasA, classB: hasB}"
  • 情况 3: 类名不确定, 只有一个 => 字符串变量
  • 情况 4: 类名不确定, 有多个 => 数组变量
  • 支持静态 class 与动态 class 的组合
<div id="test">
  <p :class="{classA:isA,classB:isB}">今天天气真不错</p>
  <p :class="arr">今天天气真不错</p>
  <p :class="arr" class="p">今天天气真不错</p>
</div>
<script src="../js/vue.js"></script>
<script>
  new Vue({
    el: "#test",
    data: {
      isA: true,
      isB: false,
      isC: true,
      arr: ["classA", "classB", "classC"],
    },
  });
</script>

批量绑定属性​

批量绑定: 一次性绑定多个动态属性
写法: v-bind="对象"
作用: 将对象中的所有属性作为动态属性都绑定给标签
使用场景: 需要传递多个属性, 而且是哪些属性是不确定的

<div id="test">
  <p v-bind="obj">111111</p>
  <p v-bind:name="obj.name">22222</p>
</div>
<script src="../js/vue.js"></script>
<script>
  new Vue({
    el: "#test",
    data: {
      time: 22,
      obj: {
        name: "晓",
        title: "今天天气真不错",
      },
    },
  });
</script>

v-on​

v-on 指令用来给标签绑定指定事件

  • 原始语法: v-on:事件名="回调函数"
  • 简化语法: @事件名="回调函数"
  • 事件回调函数, 默认接收一个 event 对象参数

methods 配置

  • 包含多个方法/函数的对象
  • 方法中的 this 被 bind 为了 vm 对象, 注意: method 不要用箭头函数
  • 一般在其中定义事件回调函数, 或者是我们自己需要调用的函数
<div id="test">
  <h1>{{count}}</h1>
  <button @click="fn($event,10)">按钮</button>
</div>
<script src="../js/vue.js"></script>
<script>
  new Vue({
    el: "#test",
    data: {
      count: 0,
    },
    methods: {
      fn($event, val) {
        console.log($event);
        this.count += val;
      },
    },
  });

methods 配置初始化的实现​

// 主要就是做2件事:
//      将methods中所有的方法都添加到vm上
//     将method中的this绑定为vm
// 实现代码:
_initMethods (methods) {
  // 遍历methods中每个method属性
  Object.keys(methods).forEach(key => {
    // this[key] = methods[key]  // 不能保证method中的this为vm
    this[key] = methods[key].bind(vm) // 指定了this为vm
  })
},
默认: 传递就是event对象
传递自定义参数:
    React: <button  onClick={() => test2('abc')}>
    Vue: <button  @click="test2('abc')">
        // 内部会在外面包一层函数  () => test2('abc')
传递自定义参数和event
    React: <button  onClick={(event) => test3('abc', event)}>
    Vue: <button  @click="test3('abc', $event)">
        // 内部会在外面包一层函数  ($event) => test3('abc', $event)
    在绑定事件时, vue提供了一个内置的变量: $event, 就是事件对象
如果事件回调, 只是执行一条更新数据的语句, 可以直接在绑定事件时直接编写

事件修饰符​

@事件后, 加.xxx 来对事件监听进行进一步处理

  • .stop: 停止事件冒泡
  • .prevent: 阻止事件的默认行为 表单
  • .once: 只执行一次 第一次执行完后, 立即移除事件

按键修饰符​

@按钮事件后, 加.xxx 来事件监听进行进一步处理

  • .keyCode: 监视的是对应的按钮 enter 键: 13
  • .enter: 监视 enter 键
<div id="test">
  <!-- 点击内部div, 不希望外部div响应点击 -->
  <div @click="alert" style="width: 200px; height: 200px; background: #888">
    <div
      @click.stop="text1"
      style="width: 100px; height: 100px; background: pink"
    ></div>
  </div>
  <!-- 不提交表单 -->
  <form action="/xxx">
    <input type="text" name="username" />
    <input type="password" name="password" />
    <input @click.prevent="text1" type="submit" value="登陆" />
  </form>
  <br />
  <button @click.once="text1">只能抢购一次</button>
  <br />
  <input
    @keyup.enter="text1"
    type="text"
    placeholder="点击enter键提示输入内容"
  />
</div>

v-model​

v-model 双向数据绑定 多用于表单收集数据

  • 能将 data 中的数据, 动态传递给 Input 显示 => 动态 value 属性
  • 能监视用户输入后, 将输入的最新值, 保存到 data 中 => input 事件监听 本质: 动态 value + input 事件监听
<input v-model="msg" />
<input :value="msg" @input="(event) => {msg = event.target.value}" />

利用 v-model 可以收集 form 中的 input / select / checkbox/ radio / textarea 等表单项的数据

<div id="test">

    <span>用户名: </span>

    <span>密码: </span>
    <input type="password" v-model="user.password" /><br />
    <span>性别: </span>
    <input type="radio" id="female" :value="0" v-model="user.sex" />
    <label for="female">女</label>
    <input type="radio" id="male" :value="1" v-model="user.sex" />
    <label for="male">男</label><br />
    <span>爱好: </span>
    <input type="checkbox" id="basket" value="basket" v-model="user.hobby" />
    <label for="basket">篮球</label>

v-model 的修饰符​

.lazy: 在失去焦点才将输入框数据同步到data中
原理: 内部绑定change事件, 而不是input事件
.number: 收集的是number类型, 而不是默认的string类型
原理: 转换为number类型后, 再同步到data中
.trim: 收集的数据自动去掉了两端的空格
原理: 输入自动收集数据时, 将两端空格trim后再保存到data中
