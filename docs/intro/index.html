docusaurus​

官方网站​

https://www.docusaurus.cn/​

快速 构建 高效 的网站，专注处理 内容​

安装 node.js​

https://nodejs.org/en/

使用安装工具 Nvm​

https://github.com/coreybutler/nvm-windows/releases​

除了直接安装外，也可以通过安装工具来安装，使用安装工具安装后更方便我们在不同的 node 版本之间进行切换，使用起来更加灵活。 这里我们以 window 下的 nvm 为例来演示，首先点击 安装 nvm， 下载最新版的 nvm-setup.exe。根据提示下一步下一步即可，同样推荐安装到默认路径。在命令行中输入 nvm version 后，出现版本即表示安装成功。

安装 node​

输入 nvm install latest 下载并安装最新版的 node

输入 nvm use latest 切换到最新版 node，输入 nvm use lts 切换到稳定版 node，也可以输入版本号来切换到指定版本。

创建一个全新的 Docusaurus 网站​

https://www.docusaurus.cn/​

在桌面上打开终端 输入指令

npx create-docusaurus@latest my-website classic

执行为完后桌面会出现一个 my-website 的文件夹

用 vs code 运行

npm start
#使用该命令可以启动项目

github 加速器​

https://github.com/dotnetcore/fastgithub/releases​

创建一个 github 仓库​

命名 用户名.github.io​

本地仓库和远程仓库建立链接​

# 初始化项目
git init
# 将项目添加到工作区（暂存）
git add .
# 将项目提交到本地仓库（已修改）
git commit -m "注释"
# 建立远程连接,使本地仓库与Git远程仓库连接
git remote add origin https://github.com/Columbine01/test.git
# 修改远程分支的名字
git branch -M gh-pages
# 将本地仓库推到远程仓库
git push -u origin gh-pages

修改项目配置​

将项目部署到 gh-pages 分支上​

# 打包项目
npm run build
# 将项目部署到github
npm run deploy
