CSS3--新增属性

怪异盒模型​

标准盒模型和怪异盒模型

  • 相同：组成结构都一样，内容，内边距，边框线，外边距
  • 不同：盒子大小计算方式不一样

设置盒模型 box-sizing

  • box-sizing：content-box 默认值，标准盒
  • box-sizing：border-box 怪异盒（IE 盒模型）

标准盒子： 总宽度：width+左右的 padding+左右的 border
怪异盒子：总宽度：width（包含 padding 和 border）

文本阴影和盒子阴影​

文本阴影（设置多个阴影，用逗号隔开）​

text-shadow：水平 垂直 模糊度 颜色

  • 水平 正右负左
  • 垂直 正下负上
p {
  text-shadow: 5px 5px 3px red;
}

盒子阴影（设置多个阴影，用逗号隔开）​

text-shadow：1 水平 2 垂直 3 模糊度 4 大小 5 颜色 6 内阴影

  • 水平 正右负左
  • 垂直 正下负上

内阴影

  • 水平：正左负右
  • 垂直：正上负下
div {
  width: 400px;
  height: 400px;
  margin: 100px auto;
  background-color: aqua;
  box-shadow: -5px -5px 10px 0px yellow inset;
}

背景属性​

背景原点​

控制背景图片开始显示的区域

background-origin 默认值是 border-box

  • border-box 边框区域显示背景图片
  • padding-box 内边距区域显示背景图片
  • content-box 内容区域显示背景图片
div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-origin: border-box; */
  /* background-origin: padding-box; */
  background-origin: border-box;
}

背景裁剪​

background-clip

  • border-box 不裁剪
  • padding-box 裁剪边框
  • content-box 裁剪内边距
div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-clip: border-box; */
  /* background-clip: padding-box; */
  /* background-clip: content-box; */
  -webkit-background-clip: text;
  color: transparent;
  /* text 把背景图裁剪到文本上 */
}

多张背景图片​

设置多重背景图片（用逗号隔开）
注意：颜色设置需要在最后一张图片设置

div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-origin: border-box; */
  /* background-origin: padding-box; */
  /* background-origin: border-box; */
}

边框属性​

圆角边框​

border-radius

  • 一个值 4 个角圆化
  • 两个值 对角方向
  • 三个值 1 左上 2 右边对角 3 右下
  • 四个值 1 左上 2 右上 3 右下 4 左下

注意：4 个角度都圆化，宽高一半

.box1 {
  height: 200px;
  width: 200px;
  border-radius: 50%;
}
.box2 {
  width: 200px;
  border-radius: 50px/20px;
  /* 一个角设置两个角度  水平半径和垂直半径 */
}

边框描边​

outline：粗细 样式 颜色
outline-offset：设置外轮廓与边框的距离，正负值都可以设置； 不是 outline 的子属性，是一个独立的属性。

div {
  width: 200px;
  height: 200px;
  border: 5px solid #bfa;
  outline: #000 dotted 5px;
  outline-offset: -5px;
}

文本属性​

文本两端对齐​

div {
  height: 200px;
  width: 200px;
  border: dotted 2px #000;
  text-align: justify;
  /* 文本两端对齐 */
  text-align-last: justify;
  /* 文本最后一行两端对齐 */
}

文本换行​

div {
  width: 200px;
  height: 200px;
  border: 2px dotted #000;
  /* word-wrap: break-word; */
  /* 智能换行  在浏览器自动挪到第二行的基础上换行*/
  word-break: break-all;
  /* 粗暴换行，只要超出就换行 */
}

文本超出​

overflow 文本超出设置

  • visible 默认值
  • hidden 隐藏
  • scroll 滚动显示 不管内容是否超出都出现滚动条组件
  • auto 滚动显示 超出才会有滚动条显示
  • overflow-x ：auto 只设置 x 轴超出滚动
  • overflow-y ：auto 只设置 y 轴超出滚动

文本溢出省略号显示​

1.设置显示范围 width
2.强制内容一行显示 white-space: nowrap;
3.超出内容隐藏 overflow: hidden;
4.隐藏内容显示省略号 text-overflow: ellipsis;

div {
  width: 150px;
  border: 1px solid #000;
  white-space: nowrap;
  /* 强制内容不换行 */
  overflow: hidden;
  /* 超出内容隐藏 */
  text-overflow: ellipsis;
  /* 隐藏内容省略号 */
}
