基本数据类型

1、基本数据类型​

  • 字符串 String

  • 数值 Number

  • 布尔值 Boolean

  • 空值 Null

  • 未定义 Undefined

  • 符号 Symbol

      包装对象：字符串，数字，布尔值它们本不算对象，但是为了给它们增加功能，指执行代码时，
               把这三类在调用方法和属性的时候包装成一个对象，执行完后把对象销毁

      undefined 和 null 只是一个值，不会变成包装对象

字符串(String)​

字符串的定义，只要用引号包裹的字符都是字符串​

var a = "孙悟空";
var b = "猪八戒";
// 检测数据是什么类型 typeof 检测数据
// typeof最后的返回值是一个字符串
console.log(typeof a); //"string"

字符串换行​

  • \ 在编辑器内可以换行 控制台不会换行
let text =
  "lorem ipsum dolor \
nsit amet, consectet";
  • \n 在控制台换行
let text = "lorem ipsum dolor \n sit amet, consectet";
  • \ 转义字符
let text = "lorem ipsum dolor \\sit amet, consectet";

字符串的 length 属性​

  • 字符串.length 可以查看字符串的长度 空格也会算上
  • 字符串[ i ] 可以查看字符串的位置，从 0 开始
var str = "   123   ";
console.log(str.length); // 9
console.log(str[0]); // ""
console.log(str[str.length - 1]); //查看最后一个字符 ""

字符串的运算​

var a = "123";
console.log(a + "hao"); //123hao
console.log(a + 1); //1231
/* 字符串和其他数据类型进行加法运算时会进行拼串操作 */

console.log(a - 10); //113
/* 字符串和其他数据类型进行加法以外的任何运算都会转换为数值再运算 */

a = "hao";
console.log(a - 10); //NaN
console.log(a * 10); //NaN
/* 除了数字以外的字符串 设置数学运算，计算不了会返回NaN */

数值(Number)​

整数​

var num = 123;
console.log(typeof num); //"number"

小数 浮点数​

  • 小数计算容易出现浮点数溢出的情况先转为整数，计算完毕后再转为小数
  • 原理：计算时先把 10 进制转换为 2 进制，计算完毕后再转为 10 进制，小数在进制转换时会出现问题
let n1 = 0.1;
let n2 = 0.2;
console.log(n1 + n2); //0.30000000000000004

常见进制数​

let a = 011; //0开头 8进制
console.log(a); //9
let b = 0x11; //0x开头 16进制
console.log(b); //17

科学计数法​

var a = 13e-5;
a = 13e5;
a = 1 / 0;
console.log(a); //Infinity 无穷

Infinity 的计算​

  • Infinity 和数字计算结果都是 Infinity
console.log(Infinity * 5); //Infinity
console.log(Infinity + 5); //Infinity
console.log(Infinity + undefined); //NaN
  • infinity 和 infinity 计算,增加都是 infinity，减少 NaN
console.log(Infinity + Infinity); //Infinity
console.log(Infinity - Infinity); //NaN
console.log(Infinity * Infinity); //Infinity
console.log(Infinity / Infinity); //NaN
  • infinity 的比较
console.log(Infinity == Infinity); //true

布尔值(Boolean)​

布尔值 false 和 true​

var a = true;
console.log(typeof a); //boolean

布尔值的计算​

console.log(true + true); //2
console.log(false + false); //0
/* 布尔值的运算 true会转换为1 false会转换为0 */

未定义(Undefined)​

var a;
console.log(a);
/* 声明变量不赋值，显示undefined*/
console.log(typeof a); //"undefined"

空值(Null)​

let b = null;
/* 开发时定义一个变量，没想好变量的值存什么，可以先设置一个null*/
var a = "";
console.log(null == a);
console.log(null == false); //false
console.log(null == undefined); //true
/* 除了null和undefined 空值和其他任何值都不相等 */
/* 
    判断一个变量是不是存在  undefined
    判断一个变量的值是不是空 null
*/

符号(Symbol)​

用来创建一个唯一的标识

let c = Symbol(); //调用Symbol（）创建了一个符号
console.log(typeof c); //"symbol"

2、引用数据类型​

  • Function 函数
  • Object 对象
  • Array 数组

3、NaN​

NaN 的计算​

  • NaN 在计算的时候，除了拼接字符串以外的操作结果都是 NaN
  • NaN 不和任何值相等，包括它自身
  • NaN 和任何值比较都会返回 false
var a = "ab";
console.log(a - 1); //NaN
console.log(NaN / undefined); //NaN
console.log(NaN * null); //NaN

isNaN 方法​

// isNaN（）作用：检测数据能不能转换数字，如果能转返回 false，不能转返回 true
console.log(isNaN("123"));

4、检测数据类型​

typeof​

可以识别基本数据类型，引用数据类型只能识别函数，无法识别对象和数组

instanceof​

不能识别出基本的数据类型 String，、Number、Boolean、Undefined、Null、Symbol,但是可以检测出引用类型，如array、object、function，

constructor​

可以检测基本数据类型，但是undefined和null检测不了

Object.prototype.toString.call​

toString方法可以检测任何的数据类型，但是不常用，执行效率低
