CSS--基础

1.css 简介​

        CSS 是层叠样式表 ( Cascading Style Sheets ) 的简称. 有时我们也会称之为 CSS 样式表或级联样式表。 CSS 也是
    一种标记语言 CSS 主要用于设置 HTML 页面中的文本内容（字体、大小、对齐方式等）、图片的外形（宽高、边框样式、边距等
    ）以及版面的布局和外观显示样式。 CSS 让我们的网页更加丰富多彩，布局更加灵活自如。简单理解：CSS 可以美化 HTML ,
    让HTML更漂亮， 让页面布局更简单。

2.css 的使用方式​

行内样式​

写在标签内部 用 style 来书写

<div style="height: 200px; width:100px ;background-color:#bfa"></div>

内部样式​

写在 head 标签内用 style 标签包裹

<style>
  div {
    width: 200px;
    height: 200px;
    background-color: #bfa;
  }
</style>

外部样式​

写在 html 文件外部，用 link 标签来引入 css 样式文件

  • rel 关联样式
  • type 文本类型
  • href css 文件的路径
<link rel="stylesheet" href="../day01/css/reset.css" />
<link rel="stylesheet" href="./css/box.css" type="text/css" />

外部导入样式​

写在 html 文件外部，

<style>
  @import url(css文件的路径);
</style>

外部样式和外部导入样式的区别​

  • link 关联加载的顺序，同时加载，@import 加载顺序，先加载 html 后加载 css
  • link 对 js 支持度更高，@import 对 js 的动 dom 支持不太好

3.css 的基本选择器​

  • 标签选择器、元素选择器（标签名）

  • id 选择器 不支持重复使用

  • class 类选择器 选择 class 名相同的元素，支持重复设置，可以设置多个名字，用空格隔开

  • 通配符选择器（*） 选择的是网页的所有 html 标签

<style>
  /* 标签选择器 */
  div{
      color:yellow
  }
  /* id选择器 */
  #idName{
      background：red
  }
  /* class类选择器 */
  .className{
      font-size:18px
  }
  /* 通配符选择器 */
  *{
    margin：0
    padding：0
  }
</style>

4.基本选择器的权重​

    选择器权重
    通配符选择器     0000
    标签选择器       0001
    class            0010
    id               0100
    行内样式         1000

important​

important 可以提升某一句 css 声明的权重
使用important声明的样式权重最高
<style>
  div {
    color: yellow !important;
  }
</style>

5.复合选择器​

交集选择器​

如果 class 名相同，标签名不同，可以通过交集选择器减少选择范围

<style>
  .yellow {
    background-color: yellow;
    width: 200px;
    height: 200px;
  }
  div.yellow {
    background-color: blue;
  }

<div class="yellow"></div>

群组选择器​

逗号连接不同的内容，设置相同的样式，简化代码

<style>
  .box1,
  .box2 {
    width: 200px;
    height: 200px;
  }
</style>

<div class="box1">box1</div>
<div class="box2">box2</div>

父子选择器​

一层嵌套，通过父元素安找到自己的子元素

  div > p {
    color: red;
  }
</style>

<div>
  <p>sun</p>
</div>

后代选择器​

多层嵌套，通过祖先找到自己的后代元素

<style>
  div p {
    color: red;
  }
</style>

<div>
  <ul>
    <li>
      <p></p>
    </li>
  </ul>
</div>

‘+’选择器

  • “+”选择器则表示某元素后相邻的兄弟元素，也就是紧挨着的，是单个的

    color: red;
  }

'~'选择器

  • 作用是查找某一个指定元素的后面的所有兄弟节点
<style>
  .h3 ~ p {
    color: red;
  }
</style>

<p>把我们的血肉筑成我们新的长城</p>

伪类选择器​

元素在不同状态下也可以设置元素的样式

<style>
  a:link {
    color: yellow;
    /* 元素访问前的样式 */
  }
  a:visted {
    color: green;
    /* 元素访问后的样式 */
  }
  a:hover {
    color: blue;
    /* 鼠标移入元素的样式 */
  }
  a:active {
    color: red;
    /* 鼠标按下的样式 */
  }

注意：其他标签只有hover和active的状态
      但是a标签4个状态都有,如果a标签的4个状态都生效
      需要按照顺序设置
      link-->visted-->hover-->active

7.css 的三大特性​

层叠：用不同的方式给内容设置不同属性，不同属性的样式会同时生效

权重：同一个内容设置相同属性，属性值不同，会发生样式冲突，权重高的样式生效

继承：父元素设置的部分属性，子元素也能继承（嵌套结构）
    核心：就近原则
    文字属性一定会继承

10.背景属性​

背景颜色​

background-color

div{
    background-color：颜色值；
}
  • 背景颜色设置透明
div{
    background-color：transparent；
}

背景图片​

background-image

div{
    background-image：url（"url"）
    /* 使用绝对或者相对地址指定背景图像 */
}

背景平铺​

background-repeat

  • no-repeat 不平铺
  • repeat 水平垂直默认平铺
  • repeat-x 水平平铺
  • repeat-y 垂直平铺
div{
    background-repeat：repeat | no-repeat | repeat-x | repeat-y
}

背景图片的位置​

background -position

  • 参数是方位名词 如果指定的两个值都是方位名词，则两个值前后顺序无关，比如 left top 和 top left 效果一致

    如果只指定了一个方位名词，另一个值省略，则第二个值默认居中对齐

    div{
        background-position：x y;
    }

  • 参数是精确单位

    如果参数值是精确坐标，那么第一个肯定是 x 坐标，第二个一定是 y 坐标。

    如果只指定一个数值，那该数值一定是 x 坐标，另一个默认垂直居中。

    如果是正数背景图将会往右、下移动，如果是负数背景图将会往左、上移动。

    div{
      background-position：200px  200px;
    }

  • 参数是混合单位

    如果指定的两个值是精确单位和方位名词混合使用，则第一个值是 x 坐标，第二个值是 y 坐标

    div{
        background-position：left 200px;
    }

  • 参数是百分比 百分比为（盒子-图片）剩余空间的百分比

    div{
        background-position：50% 50%;
    }

背景图片大小​

background-size

  • 参数是精确单位 图片的宽度和高度

    div{
      background-size：100px 100px;
    }

  • 参数是百分比 相对于盒子大小的百分比

      100% 100% 图片占满盒子，缩放比例可能会变形
      cover 等比缩放，占满盒子，图片无法完整显示
      contain 等比缩放，图片完整显示，无法占满盒子
    div{
      background-size：100% 100%;
    }

背景图片固定​

background-attachment

  • fixed 背景图像固定
  • scrol 背景图像是随着对象内容滚动
div{
    background-attachment：scroll | fixed
}

背景属性的简写​

background：颜色 图片 图片平铺方式 是否固定 图片的位置
background：transparent url（image.jpg） repeat-y fixed top;

13.元素的类型​

  • 块元素
    • 上下显示，独占一行，可以自动换行
    • 可以识别宽高，可以识别 4 个方向的 margin 和 padding
    • 块元素当做其他元素的容器
    • 特殊：p，h1~h6,dt 不要嵌套其他的块元素
  • 行内元素(span,em,strong,sub,sup,ins,del,a)
    • 水平显示，不会自动换行
    • 宽高由内容撑开，无法设置 width 和 height，可以识别水平方向的 margin 和 padding
  • 行内块元素(img,intput,textarea,select)
    • 水平显示，不会自动换行
    • 都有自己本身的宽高，可以识别设置的 width 和 height，也可以识别 4 个方向的 margin 和 padding

14.盒模型​

盒模型的组成​

盒子结构分为 4 部分​

  • 内容（width 和 height）
  • 内边距（padding）设置内容与盒子的距离
  • 边框线（border）设置盒子的边框
  • 外边距（margin）设置盒子与盒子之间的距离

内边距 padding​

  • 作用：设置盒子与内容之间的距离

  • 注意： -内边距会撑大盒子，如果设置 padding 想要总宽高不变 width 和 height 减去加上 padding 的大小 -内边距不能设置负值

  • 设置 padding

      单个方向
          padding-left
          padding-right
          padding-top
          padding-bottom
      多个方向
          padding：
          1个值，4个方向
          2个值，上下  左右
          3个值，上  左右  下
          4个值，顺时针  上  右  下  左

外边距 margin​

  • 作用：设置盒子与盒子之间的位置关系

  • margin: 0 auto

    • 让盒子水平居中
    • 注意：一般情况下 auto 只能设置水平方向的外边距自动居中，上下方向 auto 一般情况不管用，（弹性布局，定位，可以设置上下 margin 也 auto）

  • 自己移动

    • margin-left
      • 正 右
      • 负 左
    • margin-top
      • 正 下
      • 负 上

  • 设置其他盒子距离

    • margin-bottom
      • 正：让其他合适往下移动
      • 负：让其他盒子往上移动
    • margin-right
      • 正：让其他盒子往右移动
      • 负：让其他盒子往左移动

  • 外边距常见 bug

    1.两个盒子上下的外边距会重叠，谁的值大设谁
    解决：给一个盒子设置边距

    2.外边距的margin-top的 传递问题
    当父元素下的第一个子元素设置margin-top的时候，margin-top会错误传递给父元素
    解决方法：
        父元素：
           1. 阻隔作用
               border
               padding
           2. 触发BFC（是一个独立的渲染区域，有自己的布局规则）
                overflow：hidden
                display：inline-block
           3. 浮动 display：float
           4. 定位 position
           5. 弹性布局 display：flex

border：粗细 样式 颜色

  • 单一方向
    • border-left
    • border-right
    • border-top
    • border-bottom
  • 取消边框
    • border：none/0
  • 粗细
    • border-width
  • 样式
    • border-style
      • none 默认值
      • solid 实线
      • dotted 点线
      • dashed 虚线
      • double 双线
  • 颜色
    • border-color

16.浮动​

16.1浮动的作用​

  • 让块元素从上下排列变成水平排列（所有的盒子全部设置浮动）

原理

  • 浮动的盒子会脱离文档流
  • 浮动显示的范围由父元素的宽度决定，溢出的浮动元素会换行显示
  • float：left 左浮动 right 右浮动 none 不浮动

.wrap {
  width: 300px;
  height: 400px;
  border: 1px #000 solid;
}
.box1,
.box2,
.box3 {
  width: 100px;
  height: 100px;
  border: 1px solid #000;
}
.box1 {
  float: left;
  background: red;
}
.box2 {
  float: left;
  background: yellow;
}
.box3 {
  float: left;
  background-color: green;
}

<div class="wrap">
        <div class="box1"></div>
        <div class="box2"></div>
        <div class="box3"></div>
        <div class="box1"></div>
        <div class="box2"></div>
</div>

浮动的盒子会脱离文档流
如果浮动下方是正常占位的盒子，下方的盒子就会和浮动的盒子产生重叠效果（文本，图片，表单）

文本，表单，图片不会重叠

16.2清除浮动​

给受浮动影响的元素增加 clear 属性
chear： left right both清除两边浮动

16.3高度塌陷​

父元素的高度由内容撑开，如果内容浮动不占位，父元素识别不了内容的高度，就会出现高度塌陷

解决高度塌陷问题

  • 1.触发 BFC 解决（浮动区域的盒子也参与高度计算）
    overflow：hidden 推荐
    display：inline-block
    float
  • 2.在所有浮动元素的最下方设置空盒子，给空盒子设置清除浮动 clear 属性，把父元素的高度撑开
  • 3.万能清除法 用 after 把父元素的高度撑开，解决高度塌陷的问题
    .clear-fix::after {
      content: "";
      height: 0;
      display: block;
      clear: both;
      overflow: hidden;
    }
    解决 IE 低版本的兼容
    .clear-fix{
    zoom：1
    }
  • 4.在浮动元素最后面，添加一个块元素，并给元素设置clear:both

万能清除法​

用 after 把父元素的高度撑开，解决高度塌陷的问题

<div class="box1 .clear-fix">
  <div class="box2"></div>
</div>
.clear-fix::after {
  /*  content属性必须要设置 */
  content: "";
  height: 0;
  display: block;
  clear: both;
  overflow: hidden;
}
.box1 {
  border: 1px solid;
}
.box2 {
  width: 400px;
  height: 600px;
  background-color: aqua;
  float: left;
}

双伪类解决​

<div class="box1 .clear-fix">
  <div class="box2"></div>
</div>
.box1::after,
.box1::before {
  content: "";
  display: block;
}
.box1::after {
  clear: both;
}

.box1 {
  border: 1px solid;
}
.box2 {
  width: 400px;

17.定位​

定位元素可以触发五个属性

  • 4 个是位置方向 left right top bottom
  • 1 个层级关系 z-index:1 仅支持整数，可以设置负值

17.1设置定位 position​

  • 相对定位 relative
    • 占位
    • 相对于自身的原始位置
  • 绝对定位 absolute
    • 不占位，脱离文档流（完全脱离）
    • 找最近有定位属性的祖先元素，再找的就是浏览器的窗口
  • 固定定位 fixed
    • 不占位，脱离文档流（完全脱离）
    • 固定定位始终以浏览器窗口为位置移动的参照物
  • 粘性定位 sticky
    • 占位
    • 设置粘性定位，必须配合位置方向一起设置，如果没有方向粘性定位就会失效
    • 粘性定位的父元素不能设置 overflow 属性，否则粘性定位会失效
  • 取消定位 static

17.2定位居中​

第一种

/* 父元素 */
section {
  float: left;
  width: 400px;
  height: 400px;
  border: solid;
  position: relative;
}
/* 子元素 */
.box1 {
  width: 100px;
  height: 100px;
  background-color: #bfa;
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
}

第二种

/* 父元素 */
section {
  float: left;
  width: 400px;
  height: 400px;
  border: solid;
  position: relative;
}
/* 子元素 */
.box2 {
  width: 100px;
  height: 100px;
  background-color: blueviolet;
  position: absolute;
  top: 50%;
  left: 50%;
  margin-top: -50px;
  margin-left: -50px;
}
