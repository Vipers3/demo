CSS3--基础

浏览器前缀​

在 CSS3 模块标准尚未被 W3C 批准或者标准所提议的特性尚未被浏览器完全实现时，浏览器厂商会使用所谓的私有前缀来测试“试验性的”CSS 特性。各大浏览器的私有前缀如下：

  • IE 浏览器 -ms-
  • firefox 浏览器 -moz-
  • chrome 浏览器 -webkit-
  • Safari 浏览器 -webkit-
  • opera 浏览器 -o-

长度单位​

  • em 本元素字体大小的倍数
  • rem 根元素字体大小的倍数，只与根元素字体大小有关。
html {
  font-size: 200px;
}
div {
  width: 1rem;
  height: 1rem;
  background-color: #bfa;
}
  • vw 视口宽度的百分之多少 10vw 就是视口宽度的 10%。
  • vh 视口高度的百分之多少 10vh 就是视口高度的 10%。
div {
  width: 10vw;
  height: 10vh;
  background-color: #bfa;
}
  • vmax 视口宽高中大的那个的百分之多少。
  • vmin 视口宽高中小的那个的百分之多少。
div {
  width: 10vmax; /*视口宽高大的那个 10%*/
  height: 10vmin; /*视口宽高 10%*/
  background-color: cyan;
}

属性选择器​

  • E[attr] 选择具有 attr 属性的 E 元素。
  • E[attr="val"] 选择具有 attr 属性且属性值等于 val 的 E 元素。
  • E[attr~="val"] 选择具有 attr 属性且属性值为一用空格分隔的字词列表，其中一个等于 val 的 E 元素（包含只有一个值且该值等于 val 的情况。
  • E[attr^="val"] 选择具有 attr 属性且属性值为以 val 开头的字符串的 E 元素。
  • E[attr$="val"] 选择具有 attr 属性且属性值为以 val 结尾的字符串的 E 元素。
  • E[attr*="val"] 选择具有 attr 属性且属性值为包含 val 的字符串的 E 元素。
  • E[attr|="val"] 选择具有 attr 属性且属性值为以 val 开头并用连接符"-"分隔的字符串的 E 元素，如果属性值仅为 val，也将被选择。

伪类选择器​

结构伪类选择器​

  • E:first-child

      匹配父元素的第一个子元素E。

  • E:last-child

      匹配父元素的最后一个子元素E。

  • E:only-child

      匹配父元素仅有的一个子元素E。

  • E:nth-child(n)

      匹配父元素的第n个子元素E，假设该子元素不是E，则选择符无效。

  • E:nth-last-child(n)

      匹配父元素的倒数第n个子元素E，假设该子元素不是E，则选择符无效。

  • E:first-of-type

      匹配同类型中的第一个同级兄弟元素E

  • E:last-of-type

      匹配同类型中的最后一个同级兄弟元素E

  • E:only-of-type

      匹配同类型中的唯一的一个同级兄弟元素E

  • E:nth-of-type(n)

      匹配同类型中的第n个同级兄弟元素E

  • E:nth-last-of-type(n)

      匹配同类型中的倒数第n个同级兄弟元素E

否定伪类选择器​

  • E:not(s)

    排除满足括号中选择器条件的元素。

    可以配合结构伪类选择器一起用

    li:not(:nth-child(3)) {
      background: red;
    }

目标伪类选择器​

  • E:target

      匹配相关URL指向的E元素。
    h2:target {
      background-color: red;
    }

UI 状态伪类选择器​

  • E:checked

      匹配用户界面上处于选中状态的元素E。(用于input type为radio与checkbox时)

  • E:enabled

      匹配用户界面上处于可用状态的表单元素

  • E:disabled

      匹配用户界面上处于禁用状态的表单元素

伪元素选择器​

  • E:first-letter/E::first-letter

    用于给元素中的第一个文字（字母或汉字）设置样式。  设置对象内的第一个字符的样式。

  • E:first-line/E::first-line

    用于给元素中的第一行文字设置样式。

  • E:before/E::before

    动态地给元素创建一个子元素位于最开始的位置，必须设置 content 属性才生效。

  • E:after/E::after

    动态地给元素创建一个子元素位于最后的位置，必须设置 content 属性才生效。

  • E::placeholder

    用于给文本输入框或文本域设置 placeholder 文字的样式。

  • E::selection

    用于设置文本被鼠标选中之后的样式。

css 权重规则​

  1. 当多个选择器发生冲突时，会选择权重高的选择器来显示，权重越高越优先显示

  2. 比较时需要将多个选择器的权重进行相加在进行比较，如果权重一样，后面的会覆盖前面的样式

  3. 权重相加不可能超过他的最大数量级，例如无论多少个元素组成的选择器，都没有一个 class 选择器权重高

  4. 可以在样式后边添加一个！important ，这样该样式将会拥有最大的权重，其他样式都不能将其覆盖（注意：尽量不要使用！important）

怪异盒模型​

标准盒模型和怪异盒模型

  • 相同：组成结构都一样，内容，内边距，边框线，外边距
  • 不同：盒子大小计算方式不一样

设置盒模型 box-sizing

  • box-sizing：content-box 默认值，标准盒
  • box-sizing：border-box 怪异盒（IE 盒模型）

标准盒子： 总宽度：width+左右的 padding+左右的 border
怪异盒子：总宽度：width（包含 padding 和 border）

文本阴影和盒子阴影​

文本阴影（设置多个阴影，用逗号隔开）​

text-shadow：水平 垂直 模糊度 颜色

  • 水平 正右负左
  • 垂直 正下负上
p {
  text-shadow: 5px 5px 3px red;
}

盒子阴影（设置多个阴影，用逗号隔开）​

text-shadow：1 水平 2 垂直 3 模糊度 4 大小 5 颜色 6 内阴影

  • 水平 正右负左
  • 垂直 正下负上

内阴影

  • 水平：正左负右
  • 垂直：正上负下
div {
  width: 400px;
  height: 400px;
  margin: 100px auto;
  background-color: aqua;
  box-shadow: -5px -5px 10px 0px yellow inset;
}

背景属性​

背景原点​

控制背景图片开始显示的区域

background-origin 默认值是 border-box

  • border-box 边框区域显示背景图片
  • padding-box 内边距区域显示背景图片
  • content-box 内容区域显示背景图片
div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-origin: border-box; */
  /* background-origin: padding-box; */
  background-origin: border-box;
}

背景裁剪​

background-clip

  • border-box 不裁剪
  • padding-box 裁剪边框
  • content-box 裁剪内边距
div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-clip: border-box; */
  /* background-clip: padding-box; */
  /* background-clip: content-box; */
  -webkit-background-clip: text;
  color: transparent;
  /* text 把背景图裁剪到文本上 */
}

多张背景图片​

设置多重背景图片（用逗号隔开）
注意：颜色设置需要在最后一张图片设置

div {
  width: 400px;
  height: 400px;
  padding: 50px;
  border: 10px dotted #000;
  background: url("../尚品汇/image/logo.png") repeat;
  /* background-origin: border-box; */
  /* background-origin: padding-box; */
  /* background-origin: border-box; */
}

边框属性​

圆角边框​

border-radius

  • 一个值 4 个角圆化
  • 两个值 对角方向
  • 三个值 1 左上 2 右边对角 3 右下
  • 四个值 1 左上 2 右上 3 右下 4 左下

注意：4 个角度都圆化，宽高一半

.box1 {
  height: 200px;
  width: 200px;
  border-radius: 50%;
}
.box2 {
  width: 200px;
  border-radius: 50px/20px;
  /* 一个角设置两个角度  水平半径和垂直半径 */
}

边框描边​

outline：粗细 样式 颜色
outline-offset：设置外轮廓与边框的距离，正负值都可以设置； 不是 outline 的子属性，是一个独立的属性。

div {
  width: 200px;
  height: 200px;
  border: 5px solid #bfa;
  outline: #000 dotted 5px;
  outline-offset: -5px;
}

文本属性​

文本两端对齐​

div {
  height: 200px;
  width: 200px;
  border: dotted 2px #000;
  text-align: justify;
  /* 文本两端对齐 */
  text-align-last: justify;
  /* 文本最后一行两端对齐 */
}

文本换行​

div {
  width: 200px;
  height: 200px;
  border: 2px dotted #000;
  /* word-wrap: break-word; */
  /* 智能换行  在浏览器自动挪到第二行的基础上换行*/
  word-break: break-all;
  /* 粗暴换行，只要超出就换行 */
}

文本超出​

overflow 文本超出设置

  • visible 默认值
  • hidden 隐藏
  • scroll 滚动显示 不管内容是否超出都出现滚动条组件
  • auto 滚动显示 超出才会有滚动条显示
  • overflow-x ：auto 只设置 x 轴超出滚动
  • overflow-y ：auto 只设置 y 轴超出滚动

文本溢出省略号显示​

1.设置显示范围 width
2.强制内容一行显示 white-space: nowrap;
3.超出内容隐藏 overflow: hidden;
4.隐藏内容显示省略号 text-overflow: ellipsis;

div {
  width: 150px;
  border: 1px solid #000;
  white-space: nowrap;
  /* 强制内容不换行 */
  overflow: hidden;
  /* 超出内容隐藏 */
  text-overflow: ellipsis;
  /* 隐藏内容省略号 */
}

渐变​

线性渐变​

background-image：linear-gradient（方向，颜色 1，颜色 2）
方向（水平垂直） to top, to bottom, to left, to right
单词（对角方向） to left top, to 水平方向+垂直方向
角度值（deg） 正值 ：顺时针 负值 ： 逆时针

.box1 {
  background-image: linear-gradient(180deg, pink, #bfa);
}
.box1 {
  background-image: linear-gradient(rgb(240, 15, 53) 30%, blue 70%);
}
/* 
        红色实体 30%
        渐变色 70%-30%=40%
        蓝色实体 100%-70%=30%
*/
.box2 {
  background-image: linear-gradient(rgb(240, 15, 53) 40%, blue 30%);
}
/* 
        红色实体  40%
        渐变色 0%
        蓝色实体 60%
*/

径向渐变​

background-image:radial-gradient(渐变形状 大小 at 渐变中心点，start-color，...，last color)

渐变形状: 正圆 circle 椭圆 ellipse

大小:px % closest-side 最近边 farthest-side 最远边 closest-corner 最近角 farthest-corner 最远角

如果设置大小是百分比：设置两个值 水平 垂直
如果形状是正圆：大小 水平和垂直要相等

.box1 {
  background-image: radial-gradient(circle, red, yellow, blue);
}
.box2 {
  background-image: radial-gradient(circle at top right, red, yellow, blue);
}

重复渐变​

重复线性渐变
background-image：repeating-linear-gradient（方向，颜色 0，颜色 10%，颜色 20%）
重复径向渐变
background-image：repeating-radial-gradient(渐变形状 大小 at 渐变中心点，start-color，...，last color)

div {
  width: 200px;
  height: 200px;
  margin: 0 auto;
  background-image: repeating-linear-gradient(red 0, yellow 10%, green 20%);
}

过渡​

transition：1.参与过渡的属性 2.执行时间 3.延迟时间 4.速度动画

触发条件
1.需要用户动作事件触发
2.属性值需要发生改变，初始值，最终值

  • 参与过渡的属性
    transition-property：all
  • 执行时间（s/ms）
    transition-duration
  • 延迟执行
    transition-delay
  • 速度动画
    • ease 默认速度，逐渐变慢
    • linear 匀速
    • ease-in 加速
    • ease-out 减速
    • ease-in-out 先加速后减速
    • steps()自定义设置几步完成
    • cubic-bezier(0,0.91,1,0.43)设置贝塞尔曲线
div {
  width: 100px;
  height: 100px;
  background: red;
  /* transition: width 1s; */
  transition-property: width, height;
  transition-duration: 0.5s, 0.5s;
  transition-delay: 0s, 0.5s;
}
div:hover {
  width: 200px;
  height: 200px;
  background: green;
}

2D 的变换​

2D 的位移​

位移：相对定位（占位，原位置）

  • transform:translate() 同时设置水平和垂直方向的位移

  • transform:translateX(x) 设置水平方向的位移

  • transform:translateY(y) 设置垂直方向的位移

  • 简写 transform: translate(200px) x 轴位移 200px

p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: translate(200px, 200px);
}

2D 的缩放​

缩放（单位是倍数）

  • scaleX(x) 设置水平方向的缩放比例
  • scaleY(y) 设置垂直方向的缩放比例
  • scale() 同时设置水平方向和垂直方向的缩放比例
  • 支持负值，反方向缩放（倒过来）
  • 简写： transform: scale(2) 设置 x 轴和 y 轴
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: scale(2, 2);
}

2D 的旋转​

2d 属性的旋转功能（deg 角度）

  • rotate()顺时针和逆时针
  • rotateX()上下旋转
  • rotateY()左右旋转
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: rotate(360deg);
}

2D 的倾斜​

2D 的倾斜（单位是 deg 角度）

  • skewX() 元素在水平方向倾斜
  • skewY() 元素在垂直方向倾斜
  • skew() 元素在水平方向和垂直方向的倾斜
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: skew(0, 30deg);
}

3D 的变换​

开启 3D 空间​

transform-style: preserve-3d

3D 的景深​

perspective 景深，透视（加强 3d 的视觉效果，作用近大远小）

设置：

  • 父元素：每一个子元素的景深效果各不相同 perspective：1000px
  • 子元素：每个子元素的景深效果都一样 transform：perspective

3D 的位移​

3d 的位移：z 轴，正前负后

  • translateZ（z）
  • transform3d（x,y,z）第 1 个参数对应 X 轴，第 2 个参数对应 Y 轴，第 3 个参数对应 Z 轴，参数不允许省略
  • 注意：设置 3d 属性父元素需要开启 3d 空间 transform-style：preserve-3d
div {
  width: 400px;
  height: 300px;
  background: rgba(0, 0, 0, 0.4);
  border: 2px solid #000;
  margin: 100px auto;
  transform-style: preserve-3d;
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  border: solid green;
}
div:hover section {
  transform: translateZ(-300px);
}

3D 的旋转​

  • rotateX() 指定对象在 x 轴上的旋转角度
  • rotateY() 指定对象在 y 轴上的旋转角度
  • rotateZ() 指定对象在 z 轴上的旋转角度
  • rotate3d（x，y，z，角度值） x,y,z 表示最后一个参数的倍数
div {
  width: 400px;
  height: 300px;
  border: 2px solid #000;
  margin: 100px auto;
  perspective: 1000px;
  transform-style: preserve-3d;
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  background-color: green;
  transition: 2s;
}
div:hover section {
  transform: rotateZ(360deg);
}

3D 的缩放​

3d 的缩放：z 轴就是盒子厚度缩放，结合旋转一起

  • scaleZ() 指定对象的 z 轴缩放
  • scale3d() 指定对象的 3D 缩放。第 1 个参数对应 X 轴，第 2 个参数对应 Y 轴，第 3 个参数对应 Z 轴，参数不允许省略
div {
  width: 400px;
  height: 300px;
  background: rgba(0, 0, 0, 0.4);
  border: 2px solid #000;
  margin: 100px auto;
  transform-style: preserve-3d;
  transform: rotateX(15deg) rotateY(15deg);
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  border: solid green;
}
div:hover section {
  transform: scaleZ(2);
}

CSS3--关键帧和多列布局

关键帧​

关键帧的作用用来设置“由这个状态变成那个状态”。

@keyframes keyName {
  0% {
    property1: value1;
  }
  ... 100% {
    property2: value2;
  }
}

​ 0%、100%表示动画播放到 0%和 100%的时候分别希望某个元素的属性值是什么。

​ 50%，表示动画持续时间执行到 50%的时候的值。

属性​

  • animation-name 指定关键帧的名字，多个关键字用逗号隔开。

  • animation-duration 指定动画的执行时间，单位是 s。

  • animation-delay 指定对象动画的延迟时间

  • animation-timing-function

    • ​ linear 匀速
    • ​ ease
    • ​ steps()

  • animation-iteration-count 指定动画的具体循环次数

    • number:动画循环次数
    • infinite： 无限循环

  • animation-direction 设置对象动画在循环中是否反向运动

    • normal： 正常方向 (默认)
    • reverse： 反方向运行
    • alternate： 动画先正常运行再反方向运行，并持续交替运行
    • alternate-reverse： 动画先反运行再正方向运行，并持续交替运行

  • animation-fill-mode 设置对象动画时间之外的状态

    • none： 默认值。不设置对象动画之外的状态
    • forwards： 设置对象状态为动画结束时的状态
    • backwards： 设置对象状态为动画开始时的状态
    • both： 设置对象状态为动画结束或开始的状态

  • animation-play-state 设置对象动画的状态

    • running： 运动 (默认)
    • paused： 暂停

  • animation 复合属性，设置多个值

    如果只设置一个时间表示 duration，设置了两个时间分别是 duration 和 delay。其他子属性的值没有数量和顺序要求

/* 创建关键帧 */
@keyframes lld {
  0% {
  }
  100% {
    transform: translateX(600px) rotate(360deg);
  }
}

.box {
  width: 100px;
  height: 100px;
  text-align: center;
  line-height: 100px;
  background-color: cyan;

  /* 设置动画 */
  animation-name: lld;
  /* 设置动画所需的时间 */
  animation-duration: 5s;

  /* animation-delay: 3s; */

  /* 指定动画的执行次数 */
  /* animation-iteration-count: 2; */
  /* animation-iteration-count: infinite; */

  /* 设置动画的运行方向 */
  /* animation-direction: normal; */
  /* animation-direction: reverse; */
  /* animation-direction: alternate; */
  /* animation-direction: alternate-reverse; */

  /* 设置不动时候的状态1 */
  /* animation-fill-mode: forwards; */
  /* animation-fill-mode: backwards; */

  /* animation: lld .5s 2s linear infinite alternate; */
}

多列布局​

实现类似于报纸的布局方式

  • column-count 设置列数

    • auto： 根据 column-width 自定分配宽度
    • number： 用整数值来定义列数。不允许负值

  • column-gap 设置列与列之间的间隙

    • normal： 与 font-size 大小相同。假设该对象的font-size为 16px，则 normal 值为

  • column-rule: 1px dashed #ccc; 设置边框的样式

  • column-span: all 设置标题在所有内容上边

section {
  height: 500px;
  column-count: 3;
  /* 设置内容分为几列 */
  column-gap: 10px;
  /* 设置列间隙 */
  column-rule: 2px solid #f45;
  /* 列边框 */
}
h1 {
  column-span: all;
  /* 设置标题在所有内容的上边 */
  text-align: center;
}
