表格和表单

表格 table​

表格的结构​

<table>
  <!-- 表格声明 -->
  <caption></caption>
  <!-- 表格的标题 -->
  <tr>
    <!--表格的行 -->
    <td>
      <!-- 表格的列 -->
    </td>
  </tr>
</table>

表格的属性​

  • width 宽度 table/tr/td
  • height 高度 table/tr/td
  • border 边框 table
  • cellspacing 单元格与单元格之间的距离
  • cellpadding 单元格的内边距
  • align 水平对齐 table，tr，td

表格的数据分组​

为了方便后期数据的传输，让表格的结构更清晰

  • 对数据进行分组：

    • thead 表格的头部
    • tbody 表格的主体内容
    • tfoot 表格的底部

  • 注意：一个表格只能有一个 thead 和 tfoot

  • 样式区别：thead 和 tfoot 的高度由内容撑开，表格不会为其分配高度

表格的合并​

合并行

rowspan --同行合并 水平合并,合并列

合并列

colspan --同行合并 水平合并，合并列

表格合并的属性只能给单元格设置 th，td -合并的原理

一个单元格设置占位属性，让单元格多占位置，挤出多余的单元格需要删除

表单 form​

表单的属性​

  • action 表单提交的地址
  • method 表单提交的方式
    • get 不安全 数据量小
    • post 相对安全 数据量大
  • name 表单域的名字 方便后期区分表单
  • target
    • _self 当前网页覆盖提交
    • _blank 新建网页提交
<form action="http://www.baidu.com/s" method="get" target="_blank">
  <input type="text" name="wd" />
  <button>提交</button>
</form>

表单控件 input​

input 的属性

  • type 表单控件的类型
  • value input 的提交数据
  • placeholder 文本框的提示信息
  • disabled 禁用
  • maxlength 设置表单能输入数据的最大长度
  • name 数据提交的时候，必须取名字，名字由后端服务提供，不能乱写
  • checked 默认选中
  • 注意：属性和属性值相同时，属性值可以省略不写
<form action="http://www.baidu.com/s" method="get" target="_blank">
  <input
    placeholder="请你输入账号"
    type="text"
    name="wd"
    value=""
    maxlength="6"
  />
  <button>提交</button>
  <input type="checkbox" checked />唱 <input type="checkbox" />跳<br />
  <input type="radio" name="sex" checked />男
  <input type="radio" name="sex" />女<br />
</form>

表单控件的类型​

input 的 type 属性值

  • text 文本框
  • password 密码框
  • checkbox 多选框
  • radio 单选框
  • submit 提交按钮
  • reset 重置按钮
  • button 空按钮
  • file 上传文件
  • image 图片按钮
  • hidden 隐藏按钮
<form action="">
  账户：<input type="text" /><br />
  密码：<input type="password" /><br />
  <input type="submit" />
  <input type="reset" /><br />
  <input type="radio" name="sex" />男 <input type="radio" name="sex" />女<br />
  <input type="checkbox" />唱 <input type="checkbox" />跳<br />
  <input type="file" />
</form>

下拉菜单和文本域​

下拉列表的属性

  • select
    • name 名字
  • option
    • value 数据
    • disabled 禁用
    • selected 下拉列表默认选中
<form action="">
  <select name="" id="">
    <option disabled selected>--请选择收货地址</option>
    <option value="">北京</option>
    <option value="">上海</option>
  </select>
</form>

文本域

  • textarea
    • cols 宽度
    • rows 高度
  • 取消文本域的拖拽 --style="resize:none"
<textarea style="resize: none;" name="" id="" cols="30" rows="30">
请输入你的评价
</textarea>

表单字段集和标题​

fieldset 和 legend 表单字段集和标题

对表单内容分组和嵌套，让表单的结构样式更好看

<fieldset>
  <legend>文本框</legend>
  <select name="" id="">
    <option disabled selected>--请选择收货地址</option>
    <option value="">北京</option>
    <option value="">上海</option>
  </select>
</fieldset>

label 提示信息​

点击文本会选中输入框

  • for 属性 --关联表单控件 input
<!-- <label for="text">文本框：<label> <input type="text" id="text" />  -->
<!-- 简写 -->
<label>
  账户：
  <input type="text" /><br />
</label>
<label>
  密码：
  <input type="password" /><br />
</label>
