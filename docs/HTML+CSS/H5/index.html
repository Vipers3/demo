HTML5--基础

语义化标签​

  • <header> 文档或节的页眉，用作介绍性内容的容器，一个文档中可以有多个 <header> 元素。
  • <nav> 定义导航链接集合。
  • <article> 元素规定独立的自包含内容，可以独立于网站其他内容进行阅读。
  • <section> 元素定义文档中的节，可以将网站首页划分为简介、内容、联系信息等节。
  • <aside> 定义元素页面主内容之外的某些内容，比如侧栏，aside 内容应该与周围内容相关。
  • <footer> 文档或节规定页脚，包含文档的作者、版权信息、使用条款链接、联系信息等等，一个文档中可以使用多个<footer>元素。

<div class="wrap">
  <!-- 页头 -->
  <header class="top">header</header>
  <div class="center">
    <!-- 页面主导航 -->
    <nav>nav</nav>
    <!-- 主要内容 -->
    <section>
      section
      <header>header</header>
      <!-- 文章 -->
      <article>article</article>
      <footer>footer</footer>
    </section>
    <!-- 侧边栏 -->
    <aside>aside</aside>
  </div>
  <!-- 页脚 -->
  <footer class="bottom">fotter</footer>
</div>

状态标签​

meter 表示静态的度量，progress 表示动态的进度。

<meter min="0" max="100" value="30"></meter>
<progress max="100" value="50"></progress>

表单标签​

选项列表​

datalist 是选项列表的下拉内容，如果和 input 关联
datalist 设置 id 名，input 设置 list 属性的值就是 datalist 设置的 id 名
<input type="text" list="mylist" />
<datalist id="mylist">
  <option value="ww"></option>
  <option value="aa"></option>
  <option value="cc"></option>
</datalist>

文章下拉​

标签用于描述文档或文档某个部分的细节，与 <summary> 标签 配合使用可以为 details 定义标题。标题是可见的，用户点击标题时，会显示出 details。

<details>
  <summary>如何一夜暴富</summary>
  <p>洗洗睡吧</p>
</details>

拼音注释​

  • <ruby></ruby> 标签定义注释（中文注音或字符）
  • <rt></rt> 展示文字注音或字符注释。
<ruby>
  孙悟空
  <rt>sunwukun</rt>
</ruby>

关键字​

<mark></mark> 定义带有记号的文本 请在需要突出显示文本时使用,如搜索引擎搜索页面

/* 定义mark的颜色和背景颜色 */
mark {
  background-color: green;
  color: #fff;
}
<p>
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Culpa ipsa quos illum
  officiis reiciendis nostrum ex minima. Libero voluptate, maiores
  <br />laboriosam ab atque similique hic nisi iure, sed
  <mark>delectus sapiente.</mark>
</p>

表单控件​

表单控件的属性​

h5 新增的属性

  • placegolder 提示信息
  • required 设置必写字段
  • autofocus 自动聚焦
  • multiple 邮箱显示类型（输入多个值用逗号隔开）
  • pattern 可以输入正则表达式（自定义规则验证内容）
  • autocomplete 历史记录，name 属性 off 取消历史记录
  • novalidate 取消验证功能，给 form 设置
<form action="" novalidate>
  <input
    type="text"
    name="text"
    id="info"
    placeholder="请输入账号"
    autofocus
    required
  />
  <input type="submit" />
</form>

表单控件的显示类型​

<form action="">
  <!--邮箱-->
  <!--在提交的时候进行验证； 空的不会验证 -->
  <input type="email" name="" id="" />

  <!--url-->
  <!--在提交的时候进行验证； 空的不会验证 -->
  <input type="url" name="" id="" />

  <!--数字-->
  <!-- 无法输入数字以外其他字符，配合 max\min\step 属性设置数字的有效范围，提交的时候也进行验证； 空的不验证 -->
  <input type="number" name="" id="" />

  <!--搜索框-->
  <!-- 表现同 text 相同，主要是语义 -->
  <input type="search" />

  <!--电话号码 同text  但是用移动设备，会直接弹出数字键盘-->
  <input type="tel" />
  <input type="date" />
  <input type="month" />
  <input type="week" />
  <input type="time" />
  <input type="datetime-local" />
  <input type="submit" name="" id="" />
</form>

音、视频标签​

视频标签 video​

视频支持格式 3 种 mp4，ogg，webm

属性

  • src 链接资源
  • controls 播放控件
  • muted 静音
  • autoplay 自动播放（有兼容性，谷歌浏览器需要设置静音）
  • loop 重复播放
  • poster 视频封面
  • pre
  • load 预加载
<video
  src="../sources/video/vvv.mp4"
  controls
  width="200"
  poster="../sources/img/bg01.jpg"
></video>

<!-- 兼容写法 -->
<video controls autoplay muted loop>
  <source src="./sources/video/movie.mp4" type="video/mp4" />
  <source src="./sources/video/movie.ogg " type="video/ogg" />
  <source src="./sources/video/movie.webm " type="video/webm" />
</video>

音频​

音频格式支持 mp3，wav，ogg

属性

  • src 链接资源
  • controls 播放控件
  • muted 静音
  • autoplay 自动播放
  • loop 重复播放
<audio src="../sources/music/musice.mp3" controls autoplay muted></audio>

<!-- 兼容性写法 -->
<video controls>
  <source src="../sources/video/movie.mp4" type="video/mp4" />
  <source src="../sources/video/movie.webm" type="video/webm" />
  <source src="../sources/video/movie.ogg" type="video/ogg" />
  您的破浏览器不支持视频
</video>

全局属性​

  • contenteditable 规定元素内容是否可编辑。 <element contenteditable="true|false">
  • contextmenu 规定元素的上下文菜单。上下文菜单在用户点击元素时显示。 仅有Firefox可用
  • data-* 用于存储页面或应用程序的私有定制数据。
  • draggable 规定元素是否可拖动。 <element draggable="true|false|auto">
  • dropzone 规定在拖动被拖动数据时是否进行复制、移动或链接。 <element dropzone="copy|move|link">
  • hidden 规定元素隐藏或不再相关。
  • spellcheck 规定是否对元素进行拼写和语法检查。 <element spellcheck="true|false">
  • translate 规定是否应该翻译元素内容。 <element translate="yes|no">
